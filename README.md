#  Калькулятор — Лабораторная работа №2 



##  Описание
Лабораторная работа по предмету "Информационные технологии".  
Цели — 	изучить основные команды и функции **Git**, освоить практические навыки работы с **GitHub**, научиться работать с удалёнными репозиториями, ветками и основными операциями **Git**.


Реализован простой веб-калькулятор с поддержкой:
-  Сложения (`+`)
-  Умножения (`*`)
-  Сброса (`C`)
-  Выполнения операции (`=`)

##  Основные команды Git

```bash
-git clone <url>                  # клонировать репозиторий
-git add <file>                   # добавить файл в индекс
-git commit -m "сообщение"        # сделать коммит
-git push origin <branch>         # отправить изменения на GitHub
-git checkout -b <branch>         # создать и перейти в новую ветку
-git merge <branch>               # объединить ветку
-git tag -a v1.0.0 -m "описание"  # создать аннотированный тег
```

##  Вывод по лабораторной работе

В ходе выполнения лабораторной работы были освоены ключевые навыки работы с Git и платформой GitHub:
